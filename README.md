# OpenCode iOS 开发 Agent 合集

39 个中文 OpenCode 子代理，覆盖 iOS 越狱插件和 App 开发全流程。

## 安装

```bash
mkdir -p ~/.config/opencode/agents
git clone https://github.com/ac54u/agent.git /tmp/agent
cp /tmp/agent/*.md ~/.config/opencode/agents/
rm -rf /tmp/agent
```

重启 opencode 后 `@名称` 调用。
